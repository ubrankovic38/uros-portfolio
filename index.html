Feature Implementation Deep-Dives

Real production examples of turning Edition announcements into shipped code.

Challenge

Shopify announced Script deprecation with an August 2025 deadline. Multiple DigitlHaus clients relied on custom Scripts for tiered discounts, shipping rules, and payment gateways. Each script had unique business logic that needed faithful reproduction.

Solution

Built a systematic migration framework: audit existing scripts → map to Function targets (discount, shipping, payment) → implement in Rust with comprehensive test coverage → deploy via Shopify CLI with staged rollout.

Challenge

A Bemeir client needed a checkout that supported two parallel reward currencies: SeneCash (on-demand gift cards) and SeneBucks (session-scoped discount codes). The system had to be country-aware, handle distributor enrollment validation, and block checkout journeys for ineligible users — all within the new Checkout Extensibility framework.

Solution

Designed a modular Checkout UI Extension with three distinct blocks: a distributor context panel, enrollment validation gate, and rewards selector. Paired with three Rust Functions for tiered discounts, free shipping eligibility, and enrollment kit cart validation. A Go backend orchestrates Shopify GraphQL, Vertex tax, PostgreSQL, and DynamoDB.

Tech Stack
React TypeScript Rust Go GraphQL PostgreSQL DynamoDB Vertex Tax
Architecture

Async order webhooks handle reward debits with rollback on refund. Exchange rates are fetched per-session based on customer country. The Go API acts as a middleware layer between Shopify's checkout and the rewards engine.

Challenge

A client needed full control over their storefront beyond what Liquid templates could offer — custom animations, dynamic data sources, and a completely bespoke UI that couldn't be achieved within the standard Shopify theme architecture.

Solution

Built a Hydrogen storefront powered by Remix, leveraging the Storefront API with @inContext for country-aware rendering. Integrated Laravel blade templates as a hybrid approach for certain dynamic sections, deployed on Oxygen for edge performance.
